Home

The goal of this component is to provide a button to scroll back to the top of the page

Code :

<scroll-top hidden></scroll-top>

Style :

            
/* CSS style */
html {
    scroll-behavior: smooth;
}
scroll-top {
    position: fixed;
    bottom: 20px;
    right: 20px;
    width: 50px;
    height: 50px;
    background: #fff  url("assets/svg/top-arrow.svg") no-repeat center center;
    background-size: 25px 25px;
    border: 1px #000 solid;
    border-radius: 50px;
    cursor: pointer;
    transition: opacity .3s;
}

scroll-top:hover {
    transform: scale3d(1.05, 1.05, 1);
}

scroll-top:active {
    transform: scale3d(0.95, 0.95, 1);
}
