  • data-slides-visible="integer_value"
  • How many slides can be visible. Value must be higher or equal than data-slides-step value. Default value 3
  • data-slides-step="integer_value"
  • How many slides to scroll on click. Value must be less or equal than data-slides-value value. Default value 3
  • data-slides-pagination
  • Pagination under the slides
  • data-slides-infinite
  • Infinite carousel. Disable back to start and pagination
  • data-slides-ratio="width_value/height_value"
  • Define ratio of slides

First example : pagination - back to start on end - scroll step : 3 - slides visible : 3

Code :

<my-carousel data-slides-pagination data-slides-step="3" data-slides-visible="3"></my-carousel >

Preview :

Second example : infinite - scroll step : 2 - slides visible : 3

Code :

<my-carousel data-slides-infinite data-slides-pagination data-slides-step="2" data-slides-visible="3" ></my-carousel>

Preview :

Third example : do not back to start - pagination - scroll step : 1 - slides visible : 2

Code :

<my-carousel data-not-back-to-start data-slides-pagination data-slides-step="1" data-slides-visible="2"></my-carousel>

Preview :

Final example : all default

Code :

<my-carousel></my-carousel>
